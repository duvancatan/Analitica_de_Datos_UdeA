# Analítica de Datos - Instituto de Matemáticas - UdeA

Este repositorio constituye el portafolio académico y práctico del curso de Analítica de Datos del Instituto de Matemáticas de la Universidad de Antioquia. Su propósito es documentar el ciclo de vida integral del análisis de datos, consolidando competencias que van desde la ingeniería de características hasta la implementación de modelos de inteligencia artificial.
