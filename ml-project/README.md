# Analítica de Datos - Instituto de Matemáticas - UdeA

Este repositorio constituye el portafolio académico y práctico del curso de Analítica de Datos del Instituto de Matemáticas de la Universidad de Antioquia. Su propósito es documentar el ciclo de vida integral del análisis de datos, consolidando competencias que van desde la ingeniería de características hasta la implementación de modelos de inteligencia artificial.

# ML Project

Proyecto de Machine Learning para Analítica de Datos, incluyendo preprocesamiento, modelado y evaluación.

## Estructura

La estructura del proyecto es la siguiente:
- data/: datos
- notebooks/: exploración
- src/: código reproducible
- models/: modelos entrenados
- reports/: resultados

## Objetivos 

- Utilizar herramientas computacionales para la manipulación eficiente de datos.
- Aplicar técnicas de análisis exploratorio para comprender la estructura de los datos.
- Implementar modelos de aprendizaje supervisado y no supervisado.
- Evaluar e interpretar modelos estadísticos y de Machine Learning.
- Comunicar resultados de forma clara, reproducible y ética.


## Contenido del Curso


*Clase 1* :	Introducción al curso	 

*Clase 2*	:	Shell y Git	Comandos básicos de terminal, navegación, pipes, instalación y configuración de Git, repositorios y commits.

*Clase 3*	:	Git en VSCode	Integración de Git con VSCode, clonar repositorios, flujo de trabajo básico (commit, push, pull), ramas (Feature, Master, Main) y GitHub colaborativo.

*Clase 4*	:	SQL Básico	SELECT, FROM, WHERE, ORDER BY, operadores lógicos y de comparación.

*Clase 5*	:	SQL Intermedio	Joins (INNER, LEFT, RIGHT, FULL), GROUP BY, HAVING, funciones de agregación.

*Clase 6*	:	SQL Avanzado	Subconsultas, CTEs, índices, optimización básica, vistas, conexión Python ↔ SQL.

*Clase 7*	:	Presentación de proyectos	Organización de equipos, elección de dataset, definición de hipótesis y objetivos de análisis.

*Clase 8*	:	Python para datos I	numpy, pandas: estructuras de datos, lectura de archivos, limpieza básica.

*Clase 9* :	Python para datos II	Manejo de datos faltantes, merges y transformaciones, visualización con matplotlib y seaborn.

*Clase 10*: EDA I	Estadística descriptiva, análisis univariado (medidas resumen, histogramas, boxplots).

*Clase 11* : EDA II	Análisis bivariado y multivariado (correlaciones, scatterplots, heatmaps).

*Clase 12* : Proyecto EDA Avanzado	Caso práctico, storytelling de datos, elaboración de informe exploratorio.

*Clase 13* : Intro ML supervisado/no supervisado	Diferencias regresión vs. clasificación, clustering, pipeline en scikit-learn.

*Clase 14* : Regresión lineal	Fundamentos, mínimos cuadrados, LinearRegression, supuestos, R2R^2, validación cruzada.

*Clase 15* : Regresión logística	Clasificación binaria, función logística, odds ratio, métricas (accuracy, precisión, recall, F1, ROC).

*Clase 16* : Árboles de decisión	Entropía y Gini, reducción de varianza, overfitting, pruning, interpretación de reglas.

*Clase 17* : Random Forest	Bagging, ensamble de árboles, importancia de variables, evaluación.

*Clase 18* : Optimización Random Forest	Ajuste de hiperparámetros (GridSearchCV, RandomizedSearchCV), bias–variance.

*Clase 19* : Gradient Boosting	Concepto de boosting, GradientBoostingRegressor y Classifier, interpretación.

*Clase 20* : Optimización Gradient Boosting	Hiperparámetros (learning rate, n_estimators, max_depth), comparación con Random Forest, intro a XGBoost/LightGBM.

*Clase 21* : SVM	Hiperplanos y márgenes, kernels (lineal, polinomial, RBF), clasificación binaria y multiclase.

*Clase 22* : Redes Neuronales	Perceptrón multicapa, backpropagation, implementación con Keras/PyTorch, regularización.

*Clase 23* : Calibración de modelos	Curvas de calibración, Platt scaling, isotonic regression, interpretación probabilística.

*Clase 24* : Clustering I	Introducción a no supervisado, K-means, validación (inercia, silhouette).

*Clase 25* : Clustering II (Aplicación)	Clustering jerárquico, DBSCAN, casos de uso (segmentación, mercados).

*Clase 26* : Detección de anomalías	Isolation Forest, One-Class SVM, LOF, aplicaciones (fraude, calidad de datos).

*Clase 27* : Reglas de asociación	Algoritmo Apriori, métricas (soporte, confianza, lift), market basket analysis.
