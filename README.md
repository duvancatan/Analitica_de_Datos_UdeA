# CURSO ELECTIVO - ANALÍTICA DE DATOS - UdeA


*Clase 1* :	Introducción al curso	 

*Clase 2*	:	Shell y Git	Comandos básicos de terminal, navegación, pipes, instalación y configuración de Git, repositorios y commits.

*Clase 3*	:	Git en VSCode	Integración de Git con VSCode, clonar repositorios, flujo de trabajo básico (commit, push, pull), GitHub colaborativo.

*Clase 4*	:	Git Avanzado	Ramas (branching & merging), resolución de conflictos, buenas prácticas, issues y pull requests.

*Clase 5*	:	SQL Básico	SELECT, FROM, WHERE, ORDER BY, operadores lógicos y de comparación.

*Clase 6*	:	SQL Intermedio	Joins (INNER, LEFT, RIGHT, FULL), GROUP BY, HAVING, funciones de agregación.

*Clase 7*	:	SQL Avanzado	Subconsultas, CTEs, índices, optimización básica, vistas, conexión Python ↔ SQL.

*Clase 8*	:	Presentación de proyectos	Organización de equipos, elección de dataset, definición de hipótesis y objetivos de análisis.

*Clase 9*	:	Python para datos I	numpy, pandas: estructuras de datos, lectura de archivos, limpieza básica.

*Clase 10* :	Python para datos II	Manejo de datos faltantes, merges y transformaciones, visualización con matplotlib y seaborn.

*Clase 13*: EDA I	Estadística descriptiva, análisis univariado (medidas resumen, histogramas, boxplots).

*Clase 14* : EDA II	Análisis bivariado y multivariado (correlaciones, scatterplots, heatmaps).

*Clase 15* : Proyecto EDA Avanzado	Caso práctico, storytelling de datos, elaboración de informe exploratorio.

*Clase 16* : Intro ML supervisado/no supervisado	Diferencias regresión vs. clasificación, clustering, pipeline en scikit-learn.

*Clase 17* : Regresión lineal	Fundamentos, mínimos cuadrados, LinearRegression, supuestos, R2R^2, validación cruzada.

*Clase 18* : Regresión logística	Clasificación binaria, función logística, odds ratio, métricas (accuracy, precisión, recall, F1, ROC).

*Clase 19* : Árboles de decisión	Entropía y Gini, reducción de varianza, overfitting, pruning, interpretación de reglas.

*Clase 20* : Random Forest	Bagging, ensamble de árboles, importancia de variables, evaluación.

*Clase 21* : Optimización Random Forest	Ajuste de hiperparámetros (GridSearchCV, RandomizedSearchCV), bias–variance.

*Clase 22* : Gradient Boosting	Concepto de boosting, GradientBoostingRegressor y Classifier, interpretación.

*Clase 22* : Optimización Gradient Boosting	Hiperparámetros (learning rate, n_estimators, max_depth), comparación con Random Forest, intro a XGBoost/LightGBM.

*Clase 23* : SVM	Hiperplanos y márgenes, kernels (lineal, polinomial, RBF), clasificación binaria y multiclase.

*Clase 24* : Redes Neuronales	Perceptrón multicapa, backpropagation, implementación con Keras/PyTorch, regularización.

*Clase 25* : Calibración de modelos	Curvas de calibración, Platt scaling, isotonic regression, interpretación probabilística.

*Clase 26* : Clustering I	Introducción a no supervisado, K-means, validación (inercia, silhouette).

*Clase 27* : Clustering II (Aplicación)	Clustering jerárquico, DBSCAN, casos de uso (segmentación, mercados).

*Clase 28* : Detección de anomalías	Isolation Forest, One-Class SVM, LOF, aplicaciones (fraude, calidad de datos).

*Clase 29* : Reglas de asociación	Algoritmo Apriori, métricas (soporte, confianza, lift), market basket analysis.
